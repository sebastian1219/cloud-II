**# INFRA2 - EA2 Cloud Lab**



**Este repositorio despliega un laboratorio cloud en AWS Learner Lab con:**

**- VPC personalizada**

**- 2 nodos k3s en EC2**

**- NLB TCP**

**- RDS MySQL**

**- Docker Hub para imágenes**

**- Manifiestos Kubernetes**



**## 🚀 Uso**

**1. Configura los \*\*Secrets\*\* en GitHub.**

**2. Ejecuta el workflow `infra2-ea2-cloud-lab.yml`.**

**3. Valida con `scripts/validate.sh`.**

**4. Monitorea con `scripts/monitor.sh`.**

**5. Respaldos con `scripts/backup-rds.sh`.**
